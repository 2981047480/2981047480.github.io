发表于 2019-01-04

@[toc]
前几天刚配完hexo的环境并进行了页面的部署，今天将部署过程中遇到的问题做一列举。

1.找不到ssh

博主配置hexo时，老老实实跟着教程一步一步往下走的，但中间遇到了找不到ssh的问题。后来发现是没有创建ssh。所以在配置的时候，希望大家先在GitHub中配置ssh，然后返回来输入cd ~/.ssh检查是否有ssh文件夹。若成功找到ssh，则会出现下图

那么解决方法呢?最终解决方法如下
首先重新本地生成ssh-key

$ ssh-keygen -t rsa -C “邮件地址@youremail.com”

Generating public/private rsa key pair.
Enter file in which to save the key (/Users/your_user_directory/.ssh/id_rsa):<回车就好>
Enter passphrase (empty for no passphrase):<输入加密串> (不建议输入密码,hexo提交默认是无密码)

测试连接ssh -T git@github.com
